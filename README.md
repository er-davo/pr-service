# Pull Request Review System

Сервис для управления Pull Request’ами, командами и пользователями-ревьюверами. Поддерживает создание PR, назначение ревьюверов, работу с командами и выборку данных по пользователям.

# Архитектура

- Go + PostgreSQL

- Транзакционный менеджер с поддержкой reties

# Подготовка окружения

Установка зависимостей и подготовка миграций:
```bash
make setup
```

# Запуск

Поднять инфраструктуру:
```bash
docker-compose up
```

После этого сервис будет доступен на порту `8080`
